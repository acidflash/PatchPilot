Suggested Cloudflare rules:

Hostname: patch.example.com

Allow:
  URI Path starts with /api/v1/agent/

  URI Path starts with /admin
  URI Path equals /docs
  URI Path equals /redoc
  URI Path equals /openapi.json

  Agent API uses:
    X-Agent-ID: <machine_id>
    Authorization: Bearer <agent_token>

  Cloudflare Access Service Tokens
